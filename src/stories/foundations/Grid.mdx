# Responsive Layout Grid

The responsive layout grid adapts to screen size and orientation, ensuring consistency across layouts. It uses a **column-based structure** that scales from mobile to desktop.

## Key Concepts

The grid consists of three elements: **Columns**, **Gutters**, and **Margins**.

## Responsive Behavior

The grid adapts the number of columns and spacing specifications based on the breakpoint.

### 1. Mobile (< 600dp)
**4 Columns**. Best for portraits and compact lists.
*   **Columns**: 4
*   **Margins**: 16px
*   **Gutters**: 16px

### 2. Tablet (600dp - 840dp)
**8 Columns**. Used for tablet portrait and large phones.
*   **Columns**: 8
*   **Margins**: 32px
*   **Gutters**: 24px

### 3. Desktop (> 840dp)
**12 Columns**. Standard layout for web applications.
*   **Columns**: 12
*   **Margins**: Auto / Max-width
*   **Gutters**: 24px

## Breakpoint Table

| Breakpoint | Range (dp) | Columns | Gutter | Margin |
| :--- | :--- | :--- | :--- | :--- |
| **Small** | 0 - 599 | 4 | 16px | 16px |
| **Medium** | 600 - 904 | 8 | 24px | 32px |
| **Large** | 905 - 1239 | 12 | 24px | Auto |
| **X-Large** | 1240 - 1439 | 12 | 24px | Auto |
| **2X-Large** | 1440+ | 12 | 24px | Auto |

# Usage

Use the grid to align content. Content should span **columns**, not gutters.

```tsx
<div className="grid grid-cols-4 md:grid-cols-8 lg:grid-cols-12 gap-4 lg:gap-6">
  <div className="col-span-4">Main Content</div>
  <div className="col-span-4 lg:col-span-8">Side / Context</div>
</div>
```
