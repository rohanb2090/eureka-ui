# Elevation (Z-Axis)

Elevation is the relative distance between two surfaces along the z-axis. Elements with higher elevation values are visually distinguished by larger, softer shadows.

## Depth Model

In the Eureka UI Design System, all elements base themselves on a 0dp surface level (background).

-   **Depths** are measured in `dp` (density-independent pixels).
-   **Shadows** are the only visual cue for depth (we do not use Z-axis translation).

## Elevation Values

| Level | Value (dp) | Standard Shadow (CSS) | Use Case |
| :--- | :--- | :--- | :--- |
| **0** | 0dp | `none` | Background, Wallpaper |
| **1** | 1dp | `0 1px 2px 0 rgba(0,0,0,0.05)` | Table rows, card outlines |
| **2** | 3dp | `0 1px 3px 0 rgba(0,0,0,0.1)` | Standard Cards, Contained Buttons |
| **3** | 6dp | `0 4px 6px -1px rgba(0,0,0,0.1)` | Dropdown Menus, Sticky Headers |
| **4** | 8dp | `0 10px 15px -3px rgba(0,0,0,0.1)` | Modals, Navigation Drawers |
| **5** | 12dp+ | `0 20px 25px -5px rgba(0,0,0,0.1)` | Large Dialogs |
