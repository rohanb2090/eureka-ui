# Elevation (Z-Axis)

Elevation is the relative distance between two surfaces along the z-axis. Elements with higher elevation values are visually distinguished by larger, softer shadows.

## Depth Model

In the Enterprise Design System, all elements base themselves on a 0dp surface level (background).

-   **Depths** are measured in `dp` (density-independent pixels).
-   **Shadows** are the only visual cue for depth (we do not use Z-axis translation).

## Elevation Values

| Level | Value (dp) | Tailwind Class | Use Case |
| :--- | :--- | :--- | :--- |
| **0** | 0dp | `none` | Background, Wallpaper |
| **1** | 1dp | `shadow-sm` | Table rows, card outlines |
| **2** | 3dp | `shadow` | Standard Cards, Contained Buttons |
| **3** | 6dp | `shadow-md` | Dropdown Menus, Sticky Headers |
| **4** | 8dp | `shadow-lg` | Modals, Navigation Drawers |
| **5** | 12dp+ | `shadow-xl` | Large Dialogs |
