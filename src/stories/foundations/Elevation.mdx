# Elevation

Elevation uses shadows to indicate depth, hierarchy, and interactivity. Higher elevation suggests an element is closer to the user.

## Usage Guidelines

| Shadow | Value | Use Case |
| :--- | :--- | :--- |
| `shadow-sm` | `0 1px 2px 0 rgb(0 0 0 / 0.05)` | Subtle borders, table rows, pressed states. |
| `shadow` | `0 1px 3px 0 rgb(0 0 0 / 0.1), ...` | Default cards, buttons, standard UI elements. |
| `shadow-md` | `0 4px 6px -1px rgb(0 0 0 / 0.1), ...` | Dropdown menus, tooltips, hover states. |
| `shadow-lg` | `0 10px 15px -3px rgb(0 0 0 / 0.1), ...` | Modals, sticky headers, heavy floating actions. |
| `shadow-xl` | `0 20px 25px -5px rgb(0 0 0 / 0.1), ...` | Large dialogs, drawers, major overlays. |

## Usage

Use the Tailwind shadow utilities directly.

```tsx
<div className="shadow-lg">...</div>
```
