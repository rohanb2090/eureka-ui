# Iconography

We use **Lucide React** for our iconography. It provides a consistent, clean, and accessible set of SVG icons.

## Usage

```tsx
import { Mail, Settings, User } from 'lucide-react';

function Example() {
  return (
    <div className="flex gap-4">
      <Mail size={24} />
      <Settings size={24} />
      <User size={24} />
    </div>
  );
}
```
