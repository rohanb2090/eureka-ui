# Typography

**Font Family:** `IBM Plex Sans`

The typography system is designed to provide hierarchy and readability. We use **IBM Plex Sans** for all text.

## Font Weights

| Weight | CSS Value | Usage |
| :--- | :--- | :--- |
| Regular | 400 | Body text |
| Medium | 500 | Labels, secondary headings |
| Semibold | 600 | Emphasized text, buttons |
| Bold | 700 | Primary headings |
